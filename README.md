# AirControl Pro

**Smart IoT Air Conditioning Dashboard**

Dashboard web responsivo para monitoreo y control inteligente de aire acondicionado. Desarrollado como proyecto de tesis IoT.

## Características

- 🌡 Control de temperatura en tiempo real
- 💡 Gestión dinámica de dispositivos (aires y luces)
- ⚡ Estadísticas de consumo energético con gráficas
- 📅 Programación de horarios automáticos
- 🌙 Modo oscuro / claro
- 📱 Diseño responsivo: app móvil + dashboard de escritorio

## Tecnologías

- HTML5, CSS3, JavaScript (Vanilla)
- Chart.js (gráficas)
- Vite (servidor de desarrollo)
- Material Icons + Google Fonts

## Vista previa

- **Escritorio**: Sidebar lateral + grillas multi-columna
- **Móvil**: Navegación inferior + columna simple

## Ejecutar

```bash
npm install
npm run dev
```
